# Avi Load Balancer Automation Framework

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Update `config/api_config.yaml` with your chosen username and password.

## Execution
Run the main script to execute parallel test cases:
`python main.py`

## Structure
- `modules/`: Contains the API Client, Workflow Logic, and Mock Stubs.
- `config/`: YAML files for dynamic configuration.
